# Sorteio_de_Nomes
- Projeto de roleta para sorteio de nomes

## Sobre o Projeto

- Esse é uma projeto onde contém uma **roleta e um campo para colocar os nomes que deseja sortear**, com dois cliques ele vai te mostrar o nome sorteado aleatoriamente.
